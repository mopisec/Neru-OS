What is NeruOS?
HariboteOS based Hobby Operating System
First, Haribote OS is developed the hobby os that is developed through one book name "30 Nichi De Dekiru! Operating System". This books contains basic factor about developing operating system. This book has been first published more than 12 years ago. But Japan's new operating system developer is using this book to learn about hobby operating system. Isn't this really cool thing?
